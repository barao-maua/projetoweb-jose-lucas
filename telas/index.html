Sobre

Um projeto para implementar uma melhoria em uma grade de horários de aula, fornecendo para docentes e coordenadores uma maior facilidade para preencher, alterar ou excluir dados na tabela com os horários, já para alunos, é buscar uma melhor visualização dos horários de suas matérias. Deixando o mais claro possível as informações, como qual professor irá ministrar cada matéria, em qual dia da semana o aluno terá aula de alguma matéria, mas visando também deixar uma melhor e mais agradável grade de horários.
